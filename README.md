# Обработка сканов: выравнивание и обрезка

Скрипт для автоматического выравнивания (deskew) и плотной обрезки белых краёв у отсканированных фотографий.

## 📋 Требования

- **Python 3.11** (обязательно!)
- Операционная система: Windows, macOS или Linux

## 🛠️ Установка окружения

### 1. Проверьте версию Python

Убедитесь, что у вас установлен Python версии 3.11:

```bash
python --version
```

Если версия отличается, установите Python 3.11 с официального сайта: https://www.python.org/downloads/

### 2. Создайте виртуальное окружение (рекомендуется)

```bash
# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
# На Windows:
venv\Scripts\activate

# На macOS/Linux:
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

## 📁 Структура проекта

```
├── main.py              # Основной скрипт обработки
├── requirements.txt     # Зависимости Python
├── HPSCANS/             # Папка с исходными сканами (создайте вручную)
│   └── *.{jpg,jpeg,png,bmp,tiff}  # Любые изображения
└── CROPPED_PHOTOS/      # Папка с результатами (создаётся автоматически)
```

## 🚀 Использование

### 1. Подготовьте файлы

Поместите ваши отсканированные изображения в папку `HPSCANS` в корне проекта. 
Поддерживаются любые форматы: `.jpg`, `.jpeg`, `.png`, `.bmp`, `.tiff`, `.tif`

### 2. Запустите скрипт

```bash
python main.py
```

### 3. Результат

Обработанные изображения появятся в папке `CROPPED_PHOTOS/`.

## ⚙️ Настройки

В файле `main.py` можно изменить следующие параметры (строки 80-84):

```python
THRESHOLD = 230    # Порог яркости для определения белых областей (230-240)
PADDING = 5        # Отступ в пикселях от края контура (3-5 для плотной обрезки)
TIGHT_CROP = True  # True = максимальная обрезка, False = более щадящий режим
```

### Рекомендации по настройке:

- **Если остаётся белая рамка:**
  - Уменьшите `PADDING` до 3
  - Увеличьте `THRESHOLD` до 235-240

- **Если обрезаются важные части изображения:**
  - Увеличьте `PADDING` до 7-10
  - Установите `TIGHT_CROP = False`

## 📦 Зависимости

Проект использует следующие библиотеки:

- `opencv-python` — обработка изображений
- `tqdm` — прогресс-бар
- `pillow` — работа с изображениями
- `rembg` — удаление фона (опционально)
- `onnxruntime-gpu` — ускорение нейросетевых операций (GPU)

## ❗ Возможные проблемы

### Файлы не найдены
Убедитесь, что:
- Папка `HPSCANS` существует
- В папке есть изображения с расширениями `.jpg`, `.jpeg`, `.png`, `.bmp`, `.tiff`, `.tif`

### Ошибки при обработке изображений
- Проверьте, что изображения не повреждены
- Убедитесь, что все зависимости установлены

### Проблемы с производительностью
Для больших объёмов изображений убедитесь, что установлена версия `onnxruntime-gpu` и у вас есть совместимая видеокарта NVIDIA.
